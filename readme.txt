Правила заполнения админки и имён файлов:



---------------------------------------------------------------
Написание "отображаемый адрес"
---------------------------------------------------------------
Пишем полное ИМЯ БРЕНДА и МОДЕЛЬ без пробелов, тире, точек и прочих символов

========== пример ==========
agmms3
agmas35
goescan401
jouavw15



---------------------------------------------------------------
Называем файл с изображением продукта 
---------------------------------------------------------------
product - image - имя производителя-модель без пробелов, типа и прочих символов - 1.jpg


========== пример ==========
product-image-agm-ms3-1.jpg
